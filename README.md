# ader.gpt.ai

> An open-source tool for turning AI-organized knowledge into mobile-friendly study cards and content workflows.

## 소개

이 프로젝트의 목표는 긴 학습 자료를 짧고 이해하기 쉬운 카드형 콘텐츠로 바꾸는 것입니다.

기존의 자격증 공부는 PDF, 기출문제집, 메모, 요약 노트가 흩어져 있어 반복 학습이 어렵습니다. ader.gpt.ai는 이런 자료를 하나의 흐름으로 정리하고, 언제든지 모바일에서 빠르게 복습할 수 있는 학습 환경을 만드는 것을 목표로 합니다.

향후에는 그날 공부한 내용을 짧게 요약하고, 개인 블로그에 학습 기록으로 남길 수 있는 기능도 추가할 예정입니다.

## 주요 기능

- 건설안전기사 기출문제 카드화
- 핵심 개념과 암기 포인트 요약
- 문제, 정답, 해설 분리
- 모바일 친화적인 카드뉴스형 UI
- 짧은 단위의 반복 학습 구조
- AI로 정리한 자료를 웹앱 콘텐츠로 변환
- 오늘 공부한 내용을 짧은 학습 기록으로 정리

## 사용 방법

1. 저장소를 다운로드합니다.
git clone https://github.com/ader-git/ader.gpt.ai.git

2. 프로젝트 폴더로 이동합니다.
cd ader.gpt.ai

3. 필요한 패키지를 설치합니다.
npm install

4. 개발 서버를 실행합니다.
npm run dev

5. 브라우저에서 로컬 주소로 접속해 카드형 학습 콘텐츠를 확인합니다.
http://localhost:3000
